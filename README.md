# 🌎 World Explorer

Aplicación SPA en React 19 que muestra información de países del mundo.

## Tecnologías usadas
- React 19
- Vite
- React Router DOM
- Axios
- REST Countries API

## Pasos para ejecutar

```bash
npm install
npm run dev
```
